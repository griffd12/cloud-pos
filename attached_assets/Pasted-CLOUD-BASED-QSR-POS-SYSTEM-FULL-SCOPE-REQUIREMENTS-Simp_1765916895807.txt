Inheritance Rules (CRITICAL)

Configuration defined at Enterprise flows down to all Properties and RVCs

Configuration defined at Property flows only to that Property and its RVCs

Configuration defined at RVC applies only to that RVC

Overrides are allowed at Property and RVC

Enterprise configuration cannot be overridden upward

5. SEND / PICKUP CHECK & ROUND-BASED LOGIC
Rounds Concept

Each “Send” action creates a Round

Items are assigned to a round once sent

Items can only be sent once

Visual Indicator

Items already sent to kitchen show a “*”

“*” indicates:

Item has already been printed / sent to KDS

Item can never be re-sent

Send Check Behavior

Only items not yet sent are transmitted

Creates a new round record

Routes items to printers/KDS

Marks items as sent

Pickup Check

Retrieve previously open check

Sent items remain marked with “*”

Only unsent items are eligible for next Send
